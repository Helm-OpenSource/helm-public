# Helm Public Core — Launch Announcement V1

Bilingual announcement for the open-source Core. Wording is gate-relative
(Now / Next / Later, evidence gates) — do not add launch dates or time-bound
delivery promises.

## English

**🚀 Helm Public Core is now open source — Apache-2.0.**

Helm turns the judgement, evidence, review, boundary, and delivery-package work
inside enterprise AI implementation into a **forkable engineering structure**.
Generic platforms tell you how to assemble; Helm helps you keep judgement,
evidence, review, and boundaries visible — as code you can fork and verify.

Try it:

```bash
git clone https://github.com/Helm-OpenSource/helm-public.git
cd helm-public && docker compose up      # /operating · /approvals · /memory
npm run delivery:doctor                   # read-only Golden Path doctor
npm run pack:fixture-check
npm run eval:headless-signal-interface
npm run check:public-release
```

Start your first change from `extensions/case-management-sample/` — a synthetic
public sample pack (provenance under review).

**What Helm is not:** not another agent platform, LLM framework, workflow engine,
marketplace, or auto-execution plane. Recommendation ≠ commitment; review-first
throughout; no auto-send / auto-approve / auto-settle.
