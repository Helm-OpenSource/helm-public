# Helm Goal-driven Home Structure Report

## 当前首页必须优先回答的问题

1. 当前公司在打哪场仗
2. 当前最重要的 3 条推进链是什么
3. 当前最重要的 3 个阻塞是什么
4. 当前最值得拍板的 3 件事是什么
5. Helm 已经替团队推进了什么
6. 当前最应该由谁接手什么

## 本轮固定结构

当前首页固定为以下 8 个区块：

1. Current Campaign
2. Top 3 Operating Judgements
3. Top 3 Chain Moves
4. Top 3 Blockers
5. Top 3 Decision Requests
6. Helm Did
7. Role-specific Handoffs
8. Evidence / Trace entry

## 结构原则

- judgement-first
- goal / campaign first
- block / chain / decision request 前置
- 不退回对象入口堆叠
- evidence / trace 保留为 secondary layer，而不是消失

## 与 operating foundation 的关系

- Constitution 决定这页不能把 recommendation 伪装成 commitment
- Role / Audience 决定这页要明确“当前最应该由谁接手什么”
- Memory 决定这页必须保留 trace / replay 入口
- Goal / Campaign 决定这页必须先回答“当前公司在打哪场仗”
