## Conversation detail 当前最该看什么

- 当前建议怎么说

- 当前是否仍需 review-before-send

## External narrative detail 当前最该看什么

- 当前 narrative 应停在哪一层
- 当前为什么不能继续说强
- 当前是否仍要 fallback
- 当前是否 customer-visible

## 当前最该判什么

- 这句到底是 discussion-safe、boundary-only、review-before-send 还是 customer-visible
- 现在是谁接手
- 当前有没有 prerequisite / dependency / risk 还没补齐

## 当前最关键的复用话术

- 页面原文：先说 judgement，再说 reason，再说 boundary，再说 next action
- training cue：强调 `recommendation != commitment`
- acceptance cue：强调 evidence drawer 默认折叠，但 boundary 默认可见
- delivery script：强调 current scene / level、handoff reason、owner、next step 属于同一套语义

## 哪些 cue 属于页面原文

- current judgement
- why it matters
- Helm did
- decision request / collaboration request
- boundary note
- worker summary
- evidence drawer

## 哪些 cue 只属于 training / oral / acceptance / delivery

- founder 主演示时的讲解顺序
- sales / delivery 口播时的角色切换提示
- acceptance 时的风险提醒与边界核对提示
- retell template 里的复盘口径

## Delivery 结论

当前 `conversation / external narrative` 的页面原文、training cue、acceptance template、oral script 与 delivery script 已经挂在同一套 judgement / boundary / evidence 框架下，可以作为 founder demo、training、acceptance、delivery 的复用基线。
