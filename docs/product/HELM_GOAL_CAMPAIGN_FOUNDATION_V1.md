# Helm Goal / Campaign Foundation v1

## 状态

这是 Helm 当前第一轮正式的 Goal / Campaign Foundation。
它的作用不是替代 OKR / KPI 平台，而是把“当前组织正在打哪场仗”稳定地挂进 dashboard、internal operating workspace、handoff、settings 和 detail summary。

## 目标层定义

当前 Goal Layer 指的是：

1. 当前组织最重要的经营目标
2. 当前最重要的推进链
3. 当前最重要的 3 个判断
4. 当前最重要的阻塞
5. 当前最值得推动的下一步

## Campaign 层定义

当前 Campaign Layer 指的是：

1. 目标如何落到当前工作台和 internal operating workspace 的 operating mainline
2. 哪些对象、detail、handoff、approval、worker 会被这场主战役牵动
3. 哪些角色会被优先拉进来

## 当前最重要的经营主线

Helm 当前默认冻结的主战役是：

1. 保持高价值关系不掉线
2. 把 meeting-to-follow-through 接成闭环
3. 把 review / approval / handoff 前台化
4. 把 founder / sales / delivery / customer success / recruiting / partner 的跨线判断压回同一条 operating chain
5. 让 Helm 团队真的在 Helm 上经营 Helm

## 当前公司在打哪场仗

当前首页、internal operating home 和 detail chain 默认服务的是同一场仗：

1. 对外推进不能失真
2. 机会 momentum 不能掉回笔记层
3. 交付和客户成功不能只在事后补救
4. 招聘和伙伴链不能脱离经营主线
5. 边界与 sendability 不能在执行前消失

## 当前最重要的 3 个判断

当前 Goal / Campaign Layer 至少要能把这三类 judgement 放在前台：

1. 现在最该推进的关系 / 机会 / 会议后续是什么
2. 当前哪条外部表达必须继续停留在 boundary 内
3. 当前哪条 handoff 必须升级到 founder / sales / delivery / customer success / recruiting / partner

## 当前最重要的阻塞

当前默认阻塞来源包括：

1. 审批还没过
2. boundary 还不够清楚
3. 依赖对象状态没有准备好
4. 当前 detail chain 还没完成交接
5. lifecycle / payment / seat / entitlement truth 限制了新的高成本 processing

## 谁会被 Goal / Campaign 拉进来

当前默认触发规则：

1. founder
   - 当跨线优先级、重大风险、外部表达或资源仲裁需要拍板时
2. sales
   - 当机会推进、客户表达、关系 momentum 需要持续推进时
3. delivery
   - 当交付质量、预期管理、交接履约需要继续推进时
4. customer success
   - 当保留、升级、风险客户或扩张机会需要接手时
5. recruiting
   - 当关键岗位、候选人推进或 offer readiness 需要收敛时
6. partner
   - 当 custom 服务、伙伴连接和依赖风险需要接手时
7. reviewer / operator
   - 当 sendability、boundary、non-commitment、review-before-send 需要继续守住时

## Goal / Campaign 如何影响产品入口

当前至少影响：

1. dashboard
   - 当前主战役和最重要 judgement 必须先于杂项指标出现
2. internal operating workspace
   - operating home 必须先解释今天最重要的经营判断、最值得推进的 3 条链和当前决策阻塞
3. settings
   - 组织与 lifecycle / payment / seat / entitlement truth 要解释“这套系统正在保护什么经营主线”
4. unified detail navigation / handoff
   - 每段 detail 要说明它当前服务的是哪条主战役，以及下一段 handoff 请求是什么

## 当前刻意未做

当前仍然刻意未做：

1. 完整 OKR / KPI 平台
2. 完整 campaign orchestration platform
3. 完整 strategy platform

这层当前只是 Goal / Campaign Foundation，不是完整战略系统。
