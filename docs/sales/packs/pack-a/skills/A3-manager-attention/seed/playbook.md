# Skill A3 行业 SOP 骨架

## 介入触发场景（开源摘要 5 类；商业版 ≥10 类）
1. A1 识别越界承诺
2. 客户健康度异常下降
3. 关键决策点（金额阈值以上的方案洽谈）
4. 主管手工标记
5. 销售连续 N 个会议未发跟进清单

## 介入时机建议
- **会前**：A1 已发现潜在越界承诺，建议主管 1:1 销售先校准
- **同会**：客户健康度断崖式下降时建议主管旁听
- **会后**：日常风险标签建议会后复盘

## 介入方式建议
- 1:1 沟通销售（默认推荐）
- 主动加入会议（仅高风险时建议）
- 不介入但跟踪（轻度风险）

## 私密会议规则
- 销售可标记任何会议为"私密"
- A3 不显示私密会议
- 主管不知道某次会议被标记为私密
- 创始人级别可审计标记滥用率
