# 模板 01：会后确认跟进邮件

## 适用
首次接触会议结束 24 小时内发出。

## 草稿（默认建议）
```
主题：[公司] 与 [客户公司] 今日会议要点 + 后续步骤

[客户称谓]，您好：

感谢今天与我们讨论 [一句话主题]。我把今天的核心要点和后续步骤整理如下，请您查阅：

【今天讨论的关键点】
1. [客户关切 1，用客户原话]
2. [客户关切 2]
3. [客户关切 3]

【建议的后续步骤】
- [日期] 之前：[具体行动 1]（由 [我方负责人] 负责）
- [日期] 之前：[具体行动 2]

【需要进一步澄清的项】
- [关切 X 需要内部确认后给您回复，预计 Y 天内]

如有任何遗漏或调整，请直接回复本邮件。

[销售姓名]
[公司]
```

## 复核点
- 销售必须**人工修改后再发**——不允许直接发草稿
- "需要进一步澄清的项"如包含"价格 / 时间 / 范围"承诺，必须复核
